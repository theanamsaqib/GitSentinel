GitSentinel – Developer Risk Analytics

1. What it does
- Identifies risky files using:
    code churn
    change frequency
    inferred bug density
    Tracks developer impact
    Visualizes trends over time

2. Tech stack
    FastAPI
    PostgreSQL
    SQLAlchemy
    Chart.js

3. Features
    search/filter
    top-N selection
    live refresh
    drill-down per file
    time-series analytics

4. Deploy 
Use:
Render (backend)
Neon/Supabase (Postgres)
